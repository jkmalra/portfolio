## Executive Summary

Shortzy is a smaller internal tool for shortening the distance between interesting information, fast capture, and useful retrieval.

## Problem

Reading systems often fail because capture is too slow and retrieval lacks enough structure to become useful later.

## Solution

I built a compact personal system that favors low-friction capture, quick categorization, and lightweight readback views.

## Architecture

- Lightweight local storage for notes and references.
- Fast capture path with minimal required metadata.
- Readback views that prioritize recall over complexity.

## Features

- Quick save flow.
- Topic-based recall.
- Compact dashboard for recent signals.

## Roadmap

- Cross-device sync exploration.
- Better relationship views between saved signals.
